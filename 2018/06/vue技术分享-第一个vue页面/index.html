运行Engine 项目

下载项目

从gitlab中克隆项目到本地

git clone git@git.guahao-inc.com:devops/engine-web.git engine-web

运行安装项目依赖

在项目目录中，安装项目开发编译所需要的依赖

cd engine-web
npm install

运行项目

无明显报错后，运行项目

npm run dev

理解vue文件结构

模板 + 样式 + JS(模型与对模型的操作) = VUE
  • <template></template> 标签 对应着模板
  • <style></style> 标签 对应着样式
  • <script></script>便签对应着 模型与对模型的操作

模板语法

文本

数据绑定最常见的形式就是使用双大括号)的文本插值：

<span>Message: {{ msg }}</span>

使用 JavaScript 表达式

对于所有的数据绑定，Vue.js 都提供了完全的 JavaScript 表达式支持。

{{ number + 1 }}

{{ ok ? 'YES' : 'NO' }}

{{ message.split('').reverse().join('') }}

<div v-bind:id="'list-' + id"></div>

这些表达式会在所属 Vue 实例的数据作用域下作为 JavaScript 被解析。有个限制就是，每个绑定都只能包含单个表达式，所以下面的例子都不会生效。

<!-- 这是语句，不是表达式 -->
{{ var a = 1 }}

<!-- 流控制也不会生效，请使用三元表达式 -->
{{ if (ok) { return message } }}

v-if 指令

在 Vue 中，我们使用 v-if 指令实现条件渲染：

<h1 v-if="ok">Yes</h1>

也可以用 v-else 添加一个“else 块”：

<h1 v-if="ok">Yes</h1>
<h1 v-else>No</h1>

v-show 指令

另一个用于根据条件展示元素的选项是 v-show 指令。用法大致一样：

<h1 v-show="ok">Hello!</h1>

不同的是带有 v-show 的元素始终会被渲染并保留在 DOM 中。v-show 只是简单地切换元素的 CSS 属性 display

v-for指令

我们用 v-for 指令根据一组数组的选项列表进行渲染。v-for 指令需要使用 item in items 形式的特殊语法，items 是源数据数组并且 item 是数组元素迭代的别名。

<ul id="example-1">
  <li v-for="item in items">
    {{ item.message }}
  </li>
</ul>

var example1 = new Vue({
  el: '#example-1',
  data: {
    items: [
      { message: 'Foo' },
      { message: 'Bar' }
    ]
  }
})

Q&A

怎样快速入门vue？

我个人的建议是，在项目中实际操作，遇到的问题再查文档解决。

  1. 将vue教程看一遍，理解vue的语法；
  2. 将Element所有组件看一遍，知道Element提供了那些可用的组件；
  3. 动手写项目，在项目中使用。遇到vue相关的问题，基本能在vue的网站中找到答案；
  4. 遇到组件不会使用，再查看Element的文档；

前端项目，万变不离其宗，归根到底，无非就是 html、css、javascript，若想深入，还是需要学好这三大件。
